## 目錄結構

- `front-end/`: 前端資料夾，包含前端相關的程式碼和資源。
- `back-end/`: 後端資料夾，包含後端程式碼和相關資源。
- `DB/`: 資料庫資料，使用 MySQL。

## 如何操作

#### 1. 設置資料庫

從 `DB` 資料夾中獲取 `shopping_mall` 的資訊並建立資料庫與表。

#### 2.啟動後端程式

進入 `back-end` 資料夾，並執行後端程式。

#### 3.啟動前端程式

進入 `front-end` 資料夾，並安裝相依包和啟動前端程式。

#### 4.開啟前端頁面開始操作

在瀏覽器中訪問 `http://localhost:8081` 來使用應用程式。
